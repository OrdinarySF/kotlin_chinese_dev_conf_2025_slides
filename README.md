# Kotlin 全栈驱动的脑机接口与智能体未来实践

Kotlin Dev Conf China 2025 演讲幻灯片

## 内容简介

本演讲分享如何基于实际的脑机接口和智能体业务，用 Kotlin 把多语言混杂的系统收拢成一条清晰的链路。以 Zherica.AI 项目为例，讲解工程实践中的经验与解决思路。

## 运行

```bash
# 安装依赖
bun install

# 启动开发服务器
bun run dev

# 构建静态文件
bun run build

# 导出 PDF
bun run export
```
